# 📊 Sales Analytics Dashboard

## Overview
This project is an interactive Sales Analytics Dashboard built using Microsoft Power BI. It helps analyze sales performance through interactive charts, KPI cards, and filters.

## Features
- Total Sales
- Total Profit
- Total Quantity
- Sales by Region
- Sales by Category
- Sales by Segment
- Sales Trend by Order Date
- Interactive Region and Category Filters

## Tools Used
- Microsoft Power BI
- Sample Superstore Dataset
